# flasky

## Create Virtual Environment
`cd flasky`
`python3 -m venv virtual-environment-name`<br>
`python3 -m venv venv`
## Acitvate Virtual Environment
`cd flasky`
`source venv/bin/activate`
## Start Development Web Server
`export FLASK_APP=hello.py`<br>
`flask run`
## Start flask shell
`export FLASK_APP=hello.py`<br>
`flask shell`
